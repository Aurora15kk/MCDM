# MCDM
A Multi-Condition Diffusion Model for Completing Hyper-Relational Knowledge Graphs via Conditional Entity Generation


## Dependencies

  - python >= 3.8
  - pytorch >= 1.13
  - torchvision
  - pytorch-cuda=11.7
  - pip:
    - timm
    - diffusers
    - accelerate

## Max_quals

  - jf17k:
    - dataset:jf17k
    - max_quals:8

  - wikipeople:
    - dataset:wikipeople_clean
    - max_quals:10
